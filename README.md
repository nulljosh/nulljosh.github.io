# nulljosh.github.io

![version](https://img.shields.io/badge/version-v0.0.0-blue)

Personal site and small static project hub for heyitsmejosh.com.
[heyitsmejosh.com](https://heyitsmejosh.com)

## Features

- Static HTML, CSS, and JavaScript with no build step
- Main homepage at `index.html`
- GitHub Pages deploys automatically on push to `main`
- Custom domain configured via `CNAME`
- Local preview by opening `index.html`
- Pages for BC Garage Doors, reading tracker, scroll experiment, and the 1971 history project

## Run

```bash
# No build step
# Open index.html directly in a browser
```
