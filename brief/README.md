# Brief

![v2.0.0](https://img.shields.io/badge/version-2.0.0-blue)

Litigation planning tool for Trommel v. Attorney General of Canada. Charter violations from a warrantless wellness-call entry, Langley BC, August 2024.

## Features

- 7 legal grounds with case citations and damage ranges
- Outcome scenarios with probability bars
- Evidence checklist with localStorage persistence
- Limitation countdown (August 2026)
- Lawyer contact + call script
- Dark / Light theme with persistence
- PDF export via print

## Stack

- Vanilla HTML / CSS / JS — no framework, no build step
- GitHub Pages — deploys on push to main

## Structure

```
index.html     Page shell
style.css      Design tokens, layout, @media print
script.js      Grounds data, accordion, checklist, theme, PDF
```
