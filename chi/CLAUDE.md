# Chi
Feng Shui room analyzer PWA. Upload room photos, get Five Elements scoring.
Live: https://chi.heyitsmejosh.com
React + Vite + vite-plugin-pwa. Serverless rules engine (no AI/API keys).

## Quick Commands
- `./scripts/simplify.sh`
- `./scripts/monetize.sh . --write`
- `./scripts/audit.sh .`
- `./scripts/ship.sh .`
