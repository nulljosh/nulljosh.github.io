## Growth Roadmap

BCGD is the first vertical template for an AI-powered service-business operating system.

**Phase 1** (0-90 days): AI call handling + missed-call recovery + booking automation
**Phase 2** (3-9 months): Productize into repeatable package, 10-30 paying customers
**Phase 3** (9-24 months): Expand across home services verticals
**Phase 4** (24+ months): Multi-vertical AI ops platform
